# DAFNI Model Examples

This repo contains several simple examples of DAFNI ready models.

 - _[tiny-example--hello-world-model](./tiny-example--hello-world-model)_ - Simplistic linux based model showing the minimum needed for a valid model.
  Reads one environment variable and outputs a "Hello World" message to a text file.

 - _[simple-example--fibonacci-model](./simple-example--fibonacci-model)_ - Simple Python based example model. Reads in three environment variables 
  and outputs a JSON file containing a Fibonacci sequence.

 - _[rust-example--fibonacci-model](./rust-example--fibonacci-model)_ - Simple Rust based example model. Reads in three environment variables 
  and outputs a JSON file containing a Fibonacci sequence.
